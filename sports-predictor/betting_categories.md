# Categorías de Apuestas

Esta es una lista de las categorías y mercados de apuestas disponibles en la aplicación, extraídos de las imágenes de referencia. Para cada mercado se especifican las **opciones fijas** disponibles.

---

## 1. Ganador y Clasificación

| Mercado | Opciones Fijas |
|---------|----------------|
| **Resultado Final (1X2)** | `Equipo 1`, `Empate`, `Equipo 2` |
| **Equipo que se clasificará** | `Equipo 1`, `Equipo 2` |
| **Apuesta sin empate (Draw No Bet)** | `Equipo 1`, `Equipo 2` |
| **Doble Oportunidad** | `1X`, `12`, `X2` |
| **Modo de victoria en la llave** | `Equipo 1 Tiempo reglamentario`, `Equipo 1 Penaltis`, `Equipo 2 Tiempo reglamentario`, `Equipo 2 Penaltis` |

## 6. Goleador

|---------|----------------|
| **Marca Gol (Cualquier momento)** | `Marcará`, `Anotará` (por jugador) |

| **Marca al menos 2 goles** | `Sí` (por jugador) |
| **Primer/Último Goleador** | Lista de jugadores disponibles |
